Course Topics

AI Core Concepts

Generative AI, LLM applications, OpenAI API vs ChatGPT, and where agents are useful in practice.

Prompts, Tools, and Orchestration

Prompt engineering, prompt patterns, tool use, and multi-step workflow design for real tasks.

RAG, Validation, and Delivery

RAG and document flows, verification/validation, responsible AI, and presenting results in usable form.

Practical Modules

Project 1: Profile-CV Site

Result: ready public page for applications, clients, and academic/professional presentation.

Project 2: Own Solution Site

Practice module: design a website for your own automation solution and your real tasks.

Result: clear value proposition, service flow, and demo scenario for your use case.

Project 3: Task Manager

Use Cases: who needs AI agents and when

Scientist / Researcher

You work with data. Excel is not enough, heavy biostat tools are inconvenient, and waiting for a developer is too slow.

A research agent lets you ask in plain language: compare groups, find differences, show anomalies.

You focus on science, not on scripts.

Teacher

You repeatedly explain the same mistakes while checking assignments.

A grading agent finds errors and writes clear feedback for each student level.

You handle complex cases and spend time on real teaching.

Small Business / Entrepreneur

Daily chats repeat the same questions: price, size, delivery.

Your own agent knows your products and tone and replies instantly, 24/7.

You run the business instead of repeating answers.

Marketer / Sales

Posts are often published blindly: sometimes they work, sometimes not.

An agent analyzes past reactions and warns before publishing.

You validate content before release, not after failure.

Manager

Reports are long, but there is no time to read all of them.

An agent summarizes what changed, where risk is, and what needs attention.

You manage by meaning, not by document volume.

Professional / Job Seeker

You send many applications but get few replies.

An agent rewrites your experience for each vacancy without changing facts.

The employer sees a matching candidate, not a generic template.

School Student

Homework is done, but the topic is still unclear.

An agent explains differently, asks questions, and checks until understanding appears.

This is training, not cheating.

Parents

A child learns a topic and forgets it soon after.

An agent asks 2-3 short daily questions and reinforces old material.

Learning stops being done-and-forgotten.

Creative Professional

You finish creative work and wait for audience reaction.

An agent simulates feedback first: where it is unclear, boring, or engaging.

You improve before publishing and reduce disappointment.
